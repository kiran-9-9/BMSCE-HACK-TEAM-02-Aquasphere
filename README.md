## Project Overview

AquaSphere AI Dashboard is a smart water management and monitoring platform designed to provide real-time insights, incident tracking, forecasting, and administrative management through an interactive dashboard interface.

### The application includes:
- Citizen dashboard view
- Driver dashboard view
- Admin management panels
- Incident monitoring
- Forecast visualization
- Interactive map integration
- User and report management

## Setup Instructions

Follow these steps to run the project locally:

### 1. Clone the Repository

```bash
git clone <repository-link>
cd aquasphere-ai-dashboard-development
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Start Development Server

```bash
npm run dev
```

### 4. Build for Production

```bash
npm run build
```

### 5. Preview Production Build

```bash
npm run preview
```
